THIS IS A PROJECT FOR A "CANDIADATE INFORMATION AND VIDEO SUBMISSION PORTAL"
1.CLONE THE REPO
2.CREATE A DATABSE IN YOUR MONGODB AND GET ITS CONNECTION STRING
3.TO RUN THE BACKEND-
                    1.Run in terminal "cd Backend"
                    2.Run in terminal "npm install --legacy-peer-deps"
                    3.In the .env file, enter your MONGODB CONNECTION STRING
                    4.Run in terminal "npm start
4.TO RUN THE FRONTEND-
                    1.Run in terminal "cd Candidateinfo-video-portal"
                    2.Run in terminal "npm install".
                    3.Run in terminal "npm run dev"
